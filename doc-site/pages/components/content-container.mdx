The content container aligns the main content on the page depending on the viewport size, based on `breakpoints` from [design tokens](/foundation/design-tokens):

- for `small` and `medium` breakpoints, if there is side navigation, content is aligned to the left
- for the `large` breakpoint, even if there is side navigation, content is centered
- if there is no side navigation on the page, `noSidebar` prop should be passed, and content is always centered
- for the `large` breakpoint padding around content is increased from `medium` to `xlarge`
