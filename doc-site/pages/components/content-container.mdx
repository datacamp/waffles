The content container positions the main content on the page depending on the viewport size, based on `breakpoints` from [design tokens](/foundation/design-tokens):

- content is always centered, even if there is side navigation
- for all viewport sizes, maximum content width is `1200px`
- for phones, and small tablets (viewport size below `medium` breakpoint), padding around content of `medium` size is used
- for desktop, and large tablets (for viewport size above `medium` breakpoint), padding is increased to `xlarge` size
