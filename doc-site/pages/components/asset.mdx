## Props

### Asset

Assets accept all props a regular `SVGElement` would, such as `width`, `height`, `aria-` and `data-` attributes.

## Imports

You can import any existing asset as a separate component. For example to import the `AvatarPlaceholderColored` asset use:

<Imports module={['AvatarPlaceholderColored']} path="@datacamp/waffles/asset" />
