## mediaQuery

Based on the **breakpoints** defined in [design tokens](/foundation/design-tokens) return media queries for various device viewport sizes: `small`, `medium`, and `large`. When using multiple media queries make sure to write them in correct order: put the `small` before `medium`, and `medium` before `large`.

## hexToRgba

Converts **HEX** color to **RGBA** counterpart, with optional alpha setting. Mainly used to change opacity of the color.

## readableHexColor

For the best contrast and readibility, depending whether provided **HEX** color is dark or light return `tokens.colors.white` or `tokens.colors.navy` from [design tokens](/foundation/design-tokens). It is also possible to specify custom colors.

## hexColorLuminance

Calculates **HEX** color luminance. Useful for various accessibility color checks.

## isHexColor

Checks whether color is provided in **HEX** notation or not. Examples of **HEX** colors: `#05192D`, `#22E`. Examples of colors in _different_ notations: `rgb(245, 16, 0)`, `rgba(16, 8, 36, 0.4)`, `hsl(100 100% 50%)`.
